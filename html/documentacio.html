Com s'ha fet?

Introducció

En la creació d'aquest portfoli, he utilitzat un conjunt de tecnologies web essencials: HTML, CSS i una petita dosi de JavaScript. Aquesta combinació ha estat clau en l'estructuració i disseny de la pàgina, assegurant que sigui visualment atractiva i funcional. L'HTML proporciona l'estructura fonamental, el CSS ofereix l'estil i el JavaScript afegeix una capa subtil d'interactivitat per millorar l'experiència de l'usuari. Amb aquestes eines, he creat un Portfoli per mostrar efectivament el meu treball i les meves habilitats.

Tecnologies utilitzades

  • HTML (Hypertext Markup Language): Utilitzat per estructurar i organitzar el contingut del meu portfoli, incloent el disseny de la pàgina i la informació personal, així com la secció de habilitats i contacte.
  • CSS (Cascading Style Sheets): Implementat per donar estil i format visual al meu portfoli. Amb CSS, he definit els colors, les fonts, els marges i altres estils per crear una aparença coherent i atractiva.
  • JavaScript: He utilitzat JavaScript per afegir la hora en temps real i els dies que han passat desde 21 de Septembre de 2020, millorant així l'experiència de l'usuari a través de petits detalls i efectes.
